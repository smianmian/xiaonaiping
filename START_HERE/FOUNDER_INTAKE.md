## 已确认事实

1. 项目暂名：小奶瓶 / 宝宝成长记录。
2. 第一版只做 iOS 原生 App。
3. 目标用户以妈妈为主。
4. 覆盖宝宝 0-3 岁。
5. 当前替代方案主要是微信记录。
6. 痛点是不够好看，缺少温柔、有纪念册感的成长档案。
7. 第一版免费验证，先不商业化。
8. 第一版先做单人本地记录。
9. 已确认需要账号。
10. 已确认需要备份恢复。
11. 已确认服务器需要存储照片原图。
12. 已确认首发地区为香港和美国。
13. 照片可以复制进 App 私有空间。
14. 已确认疫苗提醒模板覆盖国内 + 香港。
15. 已确认崩溃上报进入第一版。
16. 账号开发期先用当前可用账号/开发者资源，正式上架前开通付费 Apple Developer 账号。
17. API 服务和对象存储区域必须在发布前按隐私和区域合规确认。
18. 崩溃上报使用 Apple 原生渠道。
19. macOS 不进入第一版。

## 合理推断

1. 高频记录体验比功能数量更重要。
2. 后端进入第一版后，仍应只做最小账号、备份恢复和照片文件存储，不做管理后台或 BI。
3. 中国大陆不进入首发，后续上线需要单独合规评估。
