# USER_STORIES.md

## 文档状态

- 项目：小奶瓶 / 宝宝成长记录
- 阶段：用户故事初版
- 日期：2026-05-28

## 已确认事实

1. 目标用户以妈妈为主。
2. 宝宝年龄覆盖 0-3 岁。
3. 当前主要替代方案是微信记录。
4. 第一版只做 iOS 单人记录。
5. 产品感觉需要温柔、有纪念册感。
6. 第一版需要账号、备份恢复和照片原图云备份。
7. 首发地区为香港和美国。
8. 疫苗提醒模板覆盖国内 + 香港。
9. 崩溃上报进入第一版。

## 合理推断

1. 用户经常处在疲惫、单手、半夜、被打断的场景。
2. 高频记录的用户故事必须比低频整理更优先。
3. 记录后的回看价值是对抗放弃记录的重要动机。

## 不进入第一版的功能

1. 家庭多人协作故事。
2. 亲友共享故事。
3. 社区发布故事。
4. 付费订阅故事。

## 角色

### 核心角色：新手妈妈

她需要在照顾宝宝的间隙快速记录，最好不费脑、不打断照护节奏，并能在之后看到这些记录形成温柔的成长档案。

### 非核心角色：其他照护者

爸爸、祖辈或保姆可能使用同一台设备查看或记录，但第一版不做独立账号和多人权限。
