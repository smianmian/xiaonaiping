## 文档状态

- 项目：小奶瓶 / 宝宝成长记录
- 阶段：发布清单更新
- 日期：2026-05-28

## 已确认事实

1. 第一版目标是 App Store 可上线的真实 iOS App。
2. 第一版免费验证。
3. 第一版首发香港和美国。
4. 第一版需要账号、备份恢复和照片原图云备份。
5. 涉及儿童、照片、成长记录和疫苗提醒，发布前必须完成隐私审查。
6. 疫苗模板覆盖国内 + 香港，崩溃上报进入第一版并使用 Apple 原生渠道。

## 合理推断

1. 上线前必须准备隐私政策和用户协议。
2. 审核说明和隐私标签必须清楚说明账号、云备份、照片原图和删除路径。
3. 中国大陆备案不阻断第一版，因为不首发中国大陆。

## 待我确认的问题

1. App Store 展示名称最终是否为“小奶瓶”。
2. 隐私政策托管地址。
3. 是否需要繁中元数据。
4. 付费 Apple Developer 账号开通时间。
5. Apple 原生崩溃上报脱敏验收方式。
6. 小奶瓶 API 正式域名。

## 不进入第一版的功能

1. 付费、订阅、内购。
2. macOS 上架。
3. 社区和分享审核材料。
4. 中国大陆提交。
