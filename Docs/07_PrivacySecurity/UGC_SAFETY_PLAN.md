# UGC_SAFETY_PLAN.md

## 文档状态

- 项目：小奶瓶 / 宝宝成长记录
- 阶段：UGC 安全判断初版
- 日期：2026-05-28

## 已确认事实

1. 第一版有用户导入宝宝照片和备注。
2. 第一版不做公开分享、社区、评论、聊天。
3. 照片默认存 App 私有空间。

## 合理推断

1. 第一版属于私有用户内容，不是公开 UGC 平台。
2. 如果未来做亲友共享或社区，必须重新设计内容安全、举报和封禁。

## 待我确认的问题

1. 是否允许导出月报图片/PDF。
2. 是否未来会做亲友分享。
3. 是否照片会上传服务器。

## 不进入第一版的功能

1. 社区。
2. 评论。
3. 聊天。
4. 公开作品广场。
5. 亲友共享链接。

## 当前安全计划

1. 用户内容默认私有。
2. 不提供公开发布入口。
3. 不需要内容审核后台。
4. 不需要举报/封禁机制。
5. 若照片上传服务器，仅用于用户本人备份/恢复，需单独审查。
