# CHINA_MAINLAND_COMPLIANCE.md

## 文档状态

- 项目：小奶瓶 / 宝宝成长记录
- 阶段：中国大陆上线合规更新
- 日期：2026-05-28
- 说明：项目管理用途，不构成正式法律意见；中国大陆上线前必须按最新规则复核

## 已确认事实

1. 小奶瓶涉及儿童、宝宝照片、家庭和成长记录。
2. 第一版需要账号、备份恢复和服务器存储照片原图。
3. 第一版首发香港和美国。
4. 中国大陆不进入第一版首发。
5. 疫苗模板覆盖国内 + 香港。
6. 第一版不做社区、公开分享、AI 诊断、订阅、电商。

## 合理推断

1. 中国大陆后续上线需要 APP 备案和 ICP 判断提前启动。
2. 如果未来使用云服务中国大陆区域，域名、隐私政策、用户协议、API 都要做备案路径判断。
3. 儿童和敏感个人信息需要更严格的告知和单独同意。
4. 照片原图云备份会提高个人信息保护和数据安全要求。
5. 国内疫苗模板后续如面向中国大陆发布，需要重新核对数据来源、文案和医疗边界。

## 待我确认的问题

1. 后续是否计划中国大陆第二阶段上线。
2. 是否有公司主体、域名和备案资源。
3. 未来中国大陆版服务器是否部署在中国大陆。
4. 后续中国大陆版是否继续使用国内 + 香港疫苗模板，及模板来源如何复核。

## 不进入第一版的功能

1. 中国大陆首发。
2. UGC 社区。
3. AI 生成或 AI 诊断。
4. 电商交易。
5. 直播/音视频平台。
6. 医疗建议。

## 中国大陆上线检查，后续阶段

| 项 | 当前结论 |
|---|---|
| APP 备案 | 第一版不需要；大陆上线前必须判断 |
| ICP 备案 | 若有大陆域名/服务器则需判断 |
| ICP 许可证 | 当前免费工具大概率暂不触发，但需按最终形态复核 |
| 公安联网备案 | 若有网站/后台/联网服务需判断 |
| 等保 | 若有后端和用户数据存储需判断 |
| 个人信息保护 | 大陆上线前必须做 |
| 儿童/未成年人条款 | 大陆上线前必须判断 |
| SDK 披露 | 大陆上线前必须做 |
| 内容安全 | 无公开 UGC，低风险；私有照片不公开 |
| 医疗资质 | 不做医疗诊断或建议 |

## 当前建议

第一版不勾选中国大陆。后续若要进入中国大陆，应单独开合规任务，确认主体、备案、域名、服务器区域、儿童数据、敏感个人信息和照片原图云备份规则。
