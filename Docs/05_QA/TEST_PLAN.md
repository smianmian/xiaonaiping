## 已确认事实

1. 第一版只测试 iOS 原生 App。
2. 必测功能包括宝宝档案、喂养、睡眠、排便、身高体重、照片时间线、疫苗提醒、月报。
3. 第一版需要账号、备份恢复和照片原图云备份。
4. 疫苗模板覆盖国内 + 香港，崩溃上报进入第一版并使用 Apple 原生渠道。
5. 敏感数据删除、权限拒绝、断网行为必须测试。

## 合理推断

1. 高频记录需要重点做疲劳场景和小屏测试。
2. 照片原图云备份、账号删除和服务器存储是高风险测试区。

## 待我确认的问题

1. 最低 iOS 版本和测试设备范围。
2. 是否需要 iPad 适配。
3. 云端数据保留周期和删除 SLA。
4. Apple 原生崩溃上报脱敏验证方式。

## 不进入第一版的功能

1. macOS 兼容测试。
2. Android / Web 测试。
3. 家庭多人协作测试。
