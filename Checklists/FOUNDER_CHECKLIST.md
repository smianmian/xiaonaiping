# 非技术创始人检查清单

开始开发前，你只需要确认这些问题：

- [x] 我能用一句话说清楚产品。
- [x] 我知道用户是谁：以新手妈妈为主。
- [x] 我知道用户现在怎么解决：主要用微信记录。
- [x] 我知道当前方案哪里不好：不好看，缺少成长档案感。
- [x] 我知道第一版只做 iOS 原生 App。
- [x] 我知道第一版不做 macOS、Android、Web。
- [x] 我知道第一版先做单人记录。
- [x] 我知道第一版免费验证，先不商业化。
- [x] 我知道哪些数据敏感：宝宝照片、生日、成长记录、疫苗提醒。
- [x] 我知道没有隐私审查不能做上传。
- [x] 我知道没有后端决策不能创建服务器或数据库。
- [x] 我确认服务器存储的目的和范围：备份恢复。
- [x] 我确认需要账号。
- [x] 我确认服务器需要存照片原图。
- [x] 我确认首发地区：香港和美国。
- [x] 我确认中国大陆不进入第一版首发。
- [x] 我确认疫苗提醒模板：国内 + 香港。
- [x] 我确认崩溃需要上报。
- [x] 我确认账号先用当前可用账号/开发者资源，后续开通付费 Apple Developer 账号。
- [x] 我确认服务器和对象存储区域：API 服务与对象存储区域需按合规策略确认。
- [ ] 我确认云端删除 SLA。
- [x] 我确认崩溃上报渠道：Apple 原生渠道。
- [ ] 我确认免费验证的量化成功指标。
- [ ] 我没有让 Codex 直接从想法跳到代码。
