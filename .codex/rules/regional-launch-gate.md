# Regional Launch Gate

没有 `Docs/08_Release/REGIONAL_LAUNCH_STRATEGY.md`，不允许进入发布计划。

由于小奶瓶涉及儿童、照片、家庭和成长记录，如果选择中国大陆上线，必须完成：

- `Docs/07_PrivacySecurity/CHINA_MAINLAND_COMPLIANCE.md`
- APP 备案判断
- ICP 备案 / ICP 许可证判断
- 个人信息保护与儿童/未成年人条款
- SDK 数据清单
- 账号删除和个人信息删除方案，如有账号
- App Store Connect 中国大陆合规材料

如果备案或资质未准备好，默认不勾选中国大陆首发。
