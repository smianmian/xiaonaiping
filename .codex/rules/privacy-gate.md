# Privacy Gate

涉及以下数据或权限时，必须先完成并更新 `Docs/07_PrivacySecurity/PRIVACY_REVIEW.md`：

- 儿童信息
- 宝宝照片
- 喂养、睡眠、排便、身高体重等成长记录
- 疫苗提醒
- 家庭成员或家庭生活数据
- 相册
- 相机
- 通知
- 账号
- 服务器存储
- iCloud / CloudKit
- 第三方 SDK

同时必须更新：

- `Docs/07_PrivacySecurity/SDK_DATA_INVENTORY.md`
- `Docs/07_PrivacySecurity/DATA_SAFETY.md`
- `Docs/07_PrivacySecurity/SECURITY_REVIEW.md`

没有隐私审查，不允许实现上传、发布或上线。
